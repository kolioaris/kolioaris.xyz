## How to clone this website
Run the following command in your terminal:
```bash
git clone https://github.com/kolioaris/kolioaris.xyz.git
```
<br/>
This will clone this repository into a new folder on your personal computer.

## How to run this website locally
First, you have to clone the repository and type this command:
```bash
cd kolioaris.xyz
```

<br/>
After that, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
